Global events

Description:
These functions are used for registering global events on all specific module istances. When the specific event is triggered on any of the specific module instances, all functions registered on the event will be called.

Quick navigation

  • $A.events.dialog.beforeOpen(func)
  • $A.events.dialog.open(func)

Global dialog events

$A.events.dialog.beforeOpen(func)

Use this function to register a new function on the beforeOpen event of all dialogs.

  • func
    Type: function
    The function to be called.
Code examples
//Creating a new Button module
var dialog = $A.newDialog();
$A.events.dialog.beforeOpen(function(dialog){
    alert('A dialog will open!');
});
dialog.open();

$A.events.dialog.open(func)

Use this function to register a new function on the open event of all dialogs.

  • func
    Type: function
    The function to be called.
Code examples
//Creating a new Button module
var dialog = $A.newDialog();
$A.events.dialog.open(function(dialog){
    alert('A dialog opened!');
});
dialog.open();
